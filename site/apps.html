학생앱 · 학부모앱

학생앱

학부모용

학부모앱

선택 기능

SEE EYES — 학습 집중 관리

학생 본인이 원할 때만 켜는 기능입니다. 앞에 둔 카메라로 지금 집중하고 있는지, 졸고 있는지, 자리를 비웠는지를 판단해 학생 본인과 관리 화면에 알려줍니다. 졸음은 본인이 제일 늦게 아는 일이라, 옆에서 한 번 짚어주는 것만으로도 달라지는 학생이 있습니다.

영상은 남지 않습니다

학부모님께서 가장 먼저 확인하실 부분이라고 생각합니다. 처리 방식은 이렇습니다.

  • ✓
    영상을 저장하지 않습니다 녹화 파일이 만들어지지 않습니다. 나중에 돌려볼 영상이 아예 존재하지 않습니다.
  • ✓
    영상을 밖으로 보내지 않습니다 분석은 학생이 쓰는 기기 안에서 즉시 이뤄집니다. 영상이 서버로 올라가지 않습니다.
  • ✓
    넘어가는 것은 상태값뿐입니다 관리 화면으로 전달되는 것은 '집중 / 졸음 / 이탈' 세 가지 판정 결과입니다. 분석에 쓰는 눈 개폐율·고개 각도·시선 방향 같은 값도 기기 안에서만 계산되고 사라집니다.
  • ✓
    끄면 그 순간 멈춥니다 카메라를 끄면 감지가 즉시 중단됩니다.
  • ✓
    안 써도 불이익이 없습니다 동의를 거부하면 SEE EYES 기능만 사용되지 않을 뿐, 다른 어떤 불이익도 없습니다. 상·벌점과도 무관합니다.
  • ✓
    언제든 철회할 수 있습니다 철회하면 감지가 중단됩니다. 다시 쓰려면 동의 절차를 처음부터 진행합니다.
