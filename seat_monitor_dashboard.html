학습 파이프라인 (사람 개입형 지도학습)

1 · 감지
태블릿이 EAR·고개각도로 이상 상태 추정 (현재 규칙 기반)
2 · 검수
사람이 실제 확인 후 오탐/졸음/이탈 판정 (정답 라벨 생성)
3 · 축적
판정 결과 + 특징값을 데이터셋으로 저장 (JSON 내보내기)
4 · 재학습
모인 데이터로 분류 모델 학습 → 정확도 개선 후 배포 (반복)
각 좌석 태블릿이 seat_status/{좌석}에 상태와 특징값(ear, pitch, duration)을 실시간 기록하면 이 대시보드가 반영합니다. 사람이 판정한 데이터는 브라우저에 저장되고(실시간 모드에서는 feedback_events 컬렉션에도 기록), JSON으로 내보내 오프라인에서 모델을 재학습할 수 있습니다.
※ 학생(미성년자) 대상 영상 분석이므로 CCTV 설치 고지·보호자 동의가 필요하며, 원본 영상보다 특징값 위주 저장을 권장합니다.
