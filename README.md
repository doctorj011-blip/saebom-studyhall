# 새봄 면학관 앱 — 배포 설명서

새봄 면학관 운영 시스템은 스마트폰 홈 화면에 앱으로 설치해서 쓰는 **웹앱(PWA)** 3종으로 구성됩니다.
모든 앱은 하나의 GitHub Pages 주소에서 배포되며, 공통 Firebase(`saebom-studyhall`)로 실시간 연동됩니다.

---

## 1. 앱 구성 & 접속 주소

배포 기본 주소: `https://doctorj011-blip.github.io/saebom-studyhall/`

| 앱 | 대상 | 주소 |
|----|------|------|
| 🧑‍🎓 **학생 앱** | 학생 | `.../saebom_student.html` |
| 👪 **학부모 앱** | 학부모 | `.../saebom_parent.html` |
| 🖥️ **관리자·키오스크 앱** | 운영자 / 입구 키오스크 | `.../saebom_schedule_with_hours.html` |

- **로그인**: 세 앱 모두 **휴대폰 번호 뒷 4자리**로 로그인합니다. (등록된 번호 기준)
- **키오스크 모드**: 관리자 앱 주소 뒤에 `#kiosk` 를 붙이면(`...saebom_schedule_with_hours.html#kiosk`) 입퇴실 전용 키오스크 화면으로 자동 실행됩니다.

## 3. 배포 / 업데이트 방법 (GitHub Pages)

앱을 수정한 뒤 아래 방법 중 하나로 올리면 몇 분 내 자동 반영됩니다.

**방법 A — 파일 교체 후 업로드 (간단)**
1. GitHub 저장소(`doctorj011-blip/saebom-studyhall`) 접속
2. 바꾼 HTML 파일을 끌어다 놓아 업로드(덮어쓰기) → **Commit**
3. 2~5분 후 배포 주소에서 새 버전 확인

**방법 B — Git 사용**
```bash
cd C:\Users\docto\saebom-studyhall
# 최신 HTML을 이 폴더로 복사한 뒤
git add -A
git commit -m "update apps"
git push
```

> ⚠️ **캐시 주의**: 학생·학부모가 이미 설치했다면, 업데이트가 바로 안 보일 수 있습니다.
> 앱을 완전히 닫았다 다시 열거나, 브라우저에서 새로고침하면 최신 버전으로 갱신됩니다.

> ℹ️ 최초 1회 설정: 저장소 **Settings → Pages → Branch: `main` / `(root)` → Save**

## 4. 학생·학부모 휴대폰에 설치하기

주소를 문자/카톡으로 보내주고, 아래 기종별 안내대로 설치하게 하세요.

### 📱 갤럭시 — 삼성 인터넷(기본 브라우저)
1. 주소를 **삼성 인터넷**으로 접속
2. 화면 **우측(하단)의 점 세 개 `⋮`** 메뉴를 탭
3. **`설치 및 바로가기`** 항목 선택
4. **설치** → 홈 화면에 앱 아이콘 생성 완료 ✅

### 📱 갤럭시/안드로이드 — Chrome
1. 주소를 **Chrome**으로 접속
2. 하단 **"앱 설치"** 배너를 탭
   - 배너가 없으면 우측 상단 **`⋮` → `앱 설치`**(또는 `홈 화면에 추가`)
3. 홈 화면에 앱 아이콘 생성 완료 ✅

### 🍎 아이폰 — Safari
1. 주소를 **Safari**로 접속 (Chrome 아님)
2. 하단 **공유 버튼 `□↑`** 탭
3. **`홈 화면에 추가`** 선택 → **추가**
4. 홈 화면에 앱 아이콘 생성 완료 ✅

> 설치하면 주소창 없이 전체화면 앱처럼 실행됩니다. 로그인은 **휴대폰 번호 뒷 4자리**.

## 5. 관리자 참고

- **관리자 앱 설치**: 위 학생 앱과 동일한 방법으로 관리자 앱 주소를 설치하면 됩니다.
- **키오스크(입구 태블릿/PC)**: 관리자 앱 주소 `#kiosk` 로 실행. 전화 뒷 4자리 입퇴실, 자동문 릴레이(Web Serial), 알림톡 발송 지원.
- **데이터 연동**: 학생 앱에서 저장한 스케줄(참여 `O` / 불참 `X`(사유) / 중간입실 `세모`(입실시간))은 Firebase를 통해 관리자 앱 시간표에 실시간 반영됩니다.
