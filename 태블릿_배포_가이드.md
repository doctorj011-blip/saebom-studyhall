# 새봄 면학관 감지 앱 · 태블릿 배포 가이드

## 1. 파일 올리기 (GitHub Pages)
아래 3개 파일을 기존 저장소(`saebom-studyhall`)에 함께 올립니다. 같은 폴더에 있어야 합니다.

- `drowsiness_detector.html` (감지 앱 본체)
- `detector_manifest.json` (PWA 설정)
- `detector_sw.js` (오프라인 캐싱)

> 카메라는 HTTPS에서만 동작합니다. GitHub Pages(`https://doctorj011-blip.github.io/...`)는 HTTPS라 문제없습니다. 로컬 파일(`file://`)로 열면 카메라가 안 켜집니다.

## 2. 태블릿에서 설치 (홈 화면에 추가)
- **안드로이드(크롬)**: 앱 주소 접속 → 우측 상단 메뉴 → "홈 화면에 추가" → 아이콘 실행
- **아이패드(사파리)**: 공유 버튼 → "홈 화면에 추가"

설치하면 주소창 없이 전체화면으로 뜹니다.

## 3. 좌석 설정
1. 앱 실행 후 상단 "좌석 번호"에 해당 좌석 번호 입력 → **설정**
2. 연결 표시등이 초록색이면 대시보드와 연결된 것
3. "카메라 시작" → 학생 얼굴이 화면에 잡히게 태블릿 위치 조정

## 4. 상시 운영 팁
- **앱 열면 자동 시작**: 체크해두면 태블릿을 켤 때마다 자동으로 감지 시작
- **화면 꺼짐 방지**: 체크(기본값). 감지 중 화면이 안 꺼짐
- **전체화면** 버튼으로 몰입 모드

## 5. 키오스크 모드 (학생이 앱 못 끄게)
- **안드로이드**: 설정 → 보안 → "화면 고정(Screen Pinning)" 켜기 → 앱에서 최근앱 버튼 길게 눌러 고정
- **아이패드**: 설정 → 손쉬운 사용 → "가이드 접근(Guided Access)" 켜기 → 앱에서 홈버튼(또는 측면버튼) 3번 눌러 시작

## 6. 대시보드로 확인
관리자는 다른 기기에서 `seat_monitor_dashboard.html` 열고 **실시간 모드** → 모든 좌석 상태가 실시간으로 표시됩니다.

## 주의 (법적)
학생(미성년자) 대상 영상 분석이므로, 실제 운영 전 **CCTV 설치 안내판 게시**와 **보호자 동의**를 받아두세요. 이 앱은 원본 영상을 저장·전송하지 않고 상태값(집중/졸음/이탈)과 특징값만 기록하도록 설계되어 있습니다.
